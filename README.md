# 🍎 Apple Inspired 3D MacBook Pro Landing Page

A modern, interactive 3D landing page inspired by Apple's product websites.  
This project focuses on creating a smooth and immersive web experience using React, Three.js and React Three Fiber.

The main goal of this project was to explore 3D web development, realistic model rendering, animations, and scroll-based interactions.

## ✨ Features

- Interactive 3D MacBook Pro model
- Smooth scroll-based animations
- 3D model rotation and screen animation
- Realistic lighting and environment setup
- Responsive design
- Modern Apple-inspired UI
- Smooth transitions and animations

## 🛠️ Tech Stack

- React.js
- Tailwind CSS
- Three.js
- React Three Fiber
- Drei
- Vite

## 📚 What I Learned

While building this project, I learned how to:

- Integrate Three.js into React applications
- Work with React Three Fiber for 3D scenes
- Load and manipulate GLTF 3D models
- Create lighting and environment effects
- Connect animations with scroll interactions
- Build responsive UI with Tailwind CSS
